## 🎮 게임 개발 학습

게임 개발의 핵심 언어인 C++과 문제 해결 능력을 기르기 위한 알고리즘, 그리고 매력적인 UI/UX를 위한 CSS를 공부하고 있습니다. 탄탄한 기본기를 바탕으로 멋진 게임을 만드는 것을 목표로 하고 있습니다.

* **C++ & 알고리즘 학습 자료**
    * [C++ 지식 공부 사이트 (YouTube)](https://www.youtube.com/watch?v=PFc4g8mxOiI&list=PL4SIC1d_ab-aOxWPucn31NHkQvNPHK1D1) - C++ 언어의 기초부터 고급 주제까지 배울 수 있는 유용한 강의 재생 목록입니다.
    * [BOJ 알고리즘 워크북](https://github.com/encrypted-def/basic-algo-lecture/blob/master/workbook.md) - 알고리즘 문제 풀이를 위한 백준 온라인 저지 워크북입니다.

* **게임 UI/UX 학습 자료**
    * [CSS 지식 공부 사이트](https://nlp-cs-quiz-app.vercel.app/) - 웹 기반 게임 UI나 엔진 내 UI 시스템의 원리를 이해하기 위해 학습하는 퀴즈 사이트입니다.
